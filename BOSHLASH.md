# 🚀 Saytni Internetga Chiqarish - Oddiy Ko'rsatma

## 📋 Nima Qilish Kerak?

Saytingizni internetga chiqarish uchun 2 ta oddiy qadam:

---

## 1️⃣ GITHUB'GA YUKLASH

### GitHub Desktop bilan (Eng Oson):

1. **Yuklab oling va o'rnating:**
   - https://desktop.github.com/
   - Dasturni oching

2. **GitHub'ga kiring:**
   - File → Options → Accounts
   - "Sign in to GitHub.com" tugmasini bosing
   - Brauzerda `alisherelmurodov88-cmd` akkaunt bilan kiring

3. **Loyihani qo'shing:**
   - File → Add Local Repository
   - "Choose..." tugmasini bosing
   - `C:\Users\user\Desktop\topshiriq` papkasini tanlang
   - "Add Repository" tugmasini bosing

4. **GitHub'ga yuklang:**
   - "Publish repository" tugmasini bosing
   - Repository name: `alishermobile`
   - Description: `Alisher Mobile - E-commerce Platform`
   - ✅ "Publish Repository" tugmasini bosing

**✅ TAYYOR!** Loyiha GitHub'da!

---

## 2️⃣ NETLIFY'DA DEPLOY QILISH

### Netlify bilan (5 daqiqa):

1. **Netlify'ga kiring:**
   - https://app.netlify.com/
   - "Sign up" tugmasini bosing
   - **"Sign up with GitHub"** ni tanlang
   - GitHub akkauntingiz bilan kiring
   - "Authorize Netlify" tugmasini bosing

2. **Yangi sayt qo'shing:**
   - "Add new site" tugmasini bosing
   - "Import an existing project" ni tanlang
   - "Deploy with GitHub" ni tanlang

3. **Repository tanlang:**
   - `alishermobile` ni toping
   - Ustiga bosing

4. **Sozlamalar (avtomatik to'ldiriladi):**
   ```
   Build command: npm run build
   Publish directory: dist
   ```
   - **"Deploy site"** tugmasini bosing

5. **Kutish:**
   - ☕ 2-3 daqiqa kuting
   - Deploy tugaydi

**✅ TAYYOR!** Sayt internetda!

---

## 🎨 SAYT MANZILINI O'ZGARTIRISH

Avtomatik berilgan manzil: `https://random-name-123456.netlify.app`

**O'zgartirish:**
1. Site settings → Domain management
2. "Change site name" tugmasini bosing
3. Yangi nom: `alisher-mobile`
4. Save

**Yangi manzil:** `https://alisher-mobile.netlify.app`

---

## 📱 TELEGRAM BOT'NI YANGILASH

Bot'ga yangi sayt manzilini qo'shing:

1. **bot/.env** faylini oching
2. **WEBAPP_URL** ni o'zgartiring:
   ```
   WEBAPP_URL=https://alisher-mobile.netlify.app
   ```
3. **Saqlang**
4. **GitHub Desktop'da:**
   - Commit message: "Bot URL yangilandi"
   - "Commit to main" → "Push origin"
