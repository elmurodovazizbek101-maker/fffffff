## 🚀 Keyingi Qadamlar

### 1. GitHub'ga Yuklash (2 daqiqa)

**GitHub Desktop bilan:**
1. Dasturni oching: https://desktop.github.com/
2. File → Add Local Repository
3. `C:\Users\user\Desktop\topshiriq` ni tanlang
4. "Publish repository" tugmasini bosing
5. Name: `alishermobile`
6. Publish

✅ **Tayyor!** GitHub'da: `github.com/alisherelmurodov88-cmd/alishermobile`

---

### 2. Netlify'da Deploy (3 daqiqa)

**Netlify bilan:**
1. Kiring: https://app.netlify.com/
2. "Sign up with GitHub"
3. "Add new site" → "Import an existing project"
4. "Deploy with GitHub"
5. `alishermobile` ni tanlang
6. "Deploy site"

✅ **Tayyor!** Sayt jonli: `https://random-name.netlify.app`

---

### 3. Domain O'zgartirish (30 soniya)

1. Site settings → Domain management
2. "Change site name"
3. Yangi nom: `alisher-mobile`
4. Save

✅ **Yangi URL:** `https://alisher-mobile.netlify.app`

---

### 4. Telegram Bot Yangilash (1 daqiqa)

1. `bot/.env` faylini oching
2. `WEBAPP_URL=https://alisher-mobile.netlify.app`
3. Saqlang va push qiling

✅ **Bot tayyor!**
