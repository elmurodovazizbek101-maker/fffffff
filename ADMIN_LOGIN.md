```

```

### 2. Admin panelga o'ting:
```
http://localhost:5174/admin/login
```

### 3. Login ma'lumotlarini kiriting:
- **Login maydoniga:** `admin`
- **Parol maydoniga:** `alisher123`

### 4. "Kirish" tugmasini bosing

## Xususiyatlar:
- ✅ Parol ko'rish/yashirish tugmasi mavjud
- ✅ Avtomatik to'ldirish o'chirilgan
- ✅ Login sahifasida ma'lumotlar ko'rsatilgan
- ✅ Xavfsiz input maydonlari
- ✅ Placeholder matnlar qo'shilgan

## Admin Panel Funksiyalari:
- 📊 Dashboard - Statistikalar
- 💰 Sales - Sotuvlar
- 📱 Products - Mahsulotlar
- 📂 Categories - Kategoriyalar  
- 💳 Debts - Qarzlar
- 👥 Customers - Mijozlar
- 🏪 Suppliers - Ta'minotchilar
- 💸 Expenses - Xarajatlar
- 👨‍💼 Employees - Xodimlar
- ⚙️ Settings - Sozlamalar
- 📷 Scanner - Skaner
- 🎯 Promotions - Reklama boshqaruvi

## Muammolar:
Agar login qilishda muammo bo'lsa:
1. Login va parolni to'g'ri kiritganingizni tekshiring
2. Brauzer keshini tozalang
3. Sahifani yangilang (F5)
