# Alisher Mobile - Online Store

O'zbekistondagi eng ishonchli mobil telefonlar do'koni.

> Note: This project must be run with Vite (`npm run dev`) or a production preview (`npm run preview`). Do not use VS Code Live Server or open `index.html` directly in the browser, because `.jsx` files require Vite's dev/build pipeline.

## 🚀 Features

- 📱 **Modern Web Interface** - React + Vite
- 🤖 **Telegram Bot Integration** - Web App support
- 🛒 **Shopping Cart** - Real-time cart management
- 👤 **User Authentication** - Telegram login
- 📊 **Admin Panel** - Complete store management
- 🌐 **Multi-language** - Uzbek, English, Russian
- 💾 **PWA Support** - Install as mobile app
- 🔔 **Push Notifications** - Order updates
- ⚡ **Offline Mode** - Service Worker caching
- 🎨 **Responsive Design** - Mobile-first approach
- 🔒 **Secure** - HTTPS, CSP headers
- ♿ **Accessible** - WCAG 2.1 AA compliant

## 🔧 Tech Stack

- **Frontend:** React 18, Vite
- **Routing:** React Router DOM v6
- **Icons:** Lucide React
- **Charts:** Recharts
- **Styling:** Modern CSS with CSS Variables
- **Telegram:** Web App API
- **PWA:** Service Worker, Web Manifest
- **SEO:** Meta tags, Open Graph, Schema.org
- **Performance:** Code splitting, lazy loading
- **Accessibility:** ARIA labels, keyboard navigation

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
